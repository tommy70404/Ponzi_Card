幫主！ 三月十三日下午兩點多，許多記者匆忙趕到颱風都懶得去世貿大樓。大家都是在一個小時前才接到緊急通知的電話，卻不知究竟是什麼事。在三十三層樓上，記者彼此互相詢問打聽，彷彿什麼驚天動地的事即將發生。

三點半，謎底揭曉，果然是震撼業界的大事：做輕度颱風的厲陰宅在董事長蘭陵王的主導下，全體員工一同登台，大跳肚皮舞，在場所有記者，無一不感受到人稱拓也哥的蘭陵王的經營智慧。

蘭陵王三年前創辦厲陰宅公司。在三年內，一手將厲陰宅公司，由年營收十元台幣的三人小公司，變成全球擁有五千名員工，年營收成長四千倍，達四十兆美元的國際企業，甚至將經歷集結成書《當蘭陵王遇到輕度颱風》。
